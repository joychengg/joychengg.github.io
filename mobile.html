Mind the Gap

Android application for users to facilitate navigation in the city of London by locating nearby subway stations the respective train schedules.
Highlights of the project requirements include:

  • Consuming the Transport for London Unified API to query train schedule and route data.
  • Processing query results using XML and JSON parsers.
  • Implementing an Android application to display the information on a London map.
