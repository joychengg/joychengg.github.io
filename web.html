InsightUBC

Git repository: https://github.com/joychengg/CourseRoomScheduling/tree/jquery2
Full stack web application for users to query available UBC courses and lecture rooms, and to schedule courses into available rooms.
Highlights of the project requirements include:

  • Parsing and serializing course data into JSON.
  • Scraping room data from UBC website into JSON using parse5 Node.js library.
  • Handling HTTP requests and responses with asynchronously logic.
  • Implementing an efficient query engine for querying large sets of course and room data.
    Queries must be executed in a timely manner to respect HTTP response timeout requirements.
  • Implementing a RESTful API using restify Node.js library.
  • Designing and implementing a web application front-end using Bootstrap, JQuery and AJAX.
